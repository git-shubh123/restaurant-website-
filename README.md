# Restaurant Website

A simple static restaurant website with pages for home, menu, about, and contact. This repository contains HTML, CSS, and JavaScript files and an image asset folder.

## Preview
Open [index.html](index.html) in your browser to view the site. For local development with correct resource loading, you can run a simple static server, for example:

```sh
python -m http.server 8000
# then open http://localhost:8000/index.html
```

## Project structure
- [index.html](index.html) — Home / landing page
- [about.html](about.html) — About the restaurant
- [menu.html](menu.html) — Menu page
- [contact.html](contact.html) — Contact / reservation page
- [style.css](style.css) — Main stylesheet
- [script.js](script.js) — Client-side JavaScript
- [README.md](README.md) — Project documentation
- [img/](img/) — Images
  - [img/category/](img/category/) — Category images
  - [img/food/](img/food/) — Food images

## Features
- Responsive layout (CSS in [style.css](style.css))
- Basic interactivity (scripts in [script.js](script.js))
- Separate pages for home, menu, about, and contact

## How to modify
- Edit page markup in the corresponding HTML files: [index.html](index.html), [menu.html](menu.html), [about.html](about.html), [contact.html](contact.html).
- Update styles in [style.css](style.css).
- Update behaviors in [script.js](script.js).
- Add or replace images inside [img/](img/) / [img/category/](img/category/) / [img/food/](img/food/).
